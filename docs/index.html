Fejlesztői dokumentáció

Laravel 10 alapú TTK ügyintézési rendszer technikai és architekturális bemutatása

Laravel 10.46.0 PHP 8.1+ Moduláris Enterprise

Dokumentáció áttekintés

Projekt összefoglaló: A TTK ügyintézési rendszer egy Laravel 10.46.0 LTS alapú enterprise alkalmazás, amely modern fejlesztési elveket követ és moduláris architektúrát használ. A rendszer támogatja a munkafolyamat-kezelést, jogosultságkezelést, integrált PDF/Excel generálást és IMAP alapú email integrációt.

Technológiai alapok

  • Laravel 10.46.0 LTS
  • PHP 8.1+ strict typing
  • MySQL/MariaDB adatbázis
  • Redis cache és session
  • jQuery frontend
  • Composer dependency management

Architekturális jellemzők

  • Moduláris MVC felépítés
  • Event-driven architecture
  • Service layer pattern
  • Repository pattern
  • Domain-driven design
  • SOLID principles

Kulcs funkciók

  • Munkafolyamat-kezelés
  • Role-based hozzáférés
  • PDF/Excel generálás
  • IMAP email integráció
  • REST API endpoints
  • Automatikus deployment

Enterprise funkciók

  • Multi-guard authentication
  • Permission-based authorization
  • Audit logging
  • Background job processing
  • Cache optimalizáció
  • Monitoring és alerting

Quick Start:
  1. Klónozd a repository-t: git clone https://github.com/awaxo/workflowTTK.git
  2. Telepítsd a függőségeket: composer install && npm install
  3. Másold és konfigurálj: cp .env.example .env
  4. Generálj kulcsot: php artisan key:generate
  5. Futtasd a migrációkat: php artisan migrate:fresh --seed
  6. Indítsd el a dev szervert: php artisan serve
